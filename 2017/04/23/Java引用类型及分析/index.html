Java引用类型及分析

Java引用类型及分析

今天学习了java的引用类型和他的具体是在jvm垃圾回收中是如何使用的。强引用就是一般的引用，其他的引用都在java.lang.ref 包中

Java 引用类型分类

  • 强引用
  • 软引用
  • 弱引用
  • 虚引用

强引用

就是我们一般用java写引用的类型比如说

Object o = new Object();

这个就是强引用类型的一个例子.大概分析一下。先声明一个Object的变量o分配在Java stack，然后变量o指向引用由Java heap分配的一个Object对象

强引用的具备以下特点

  • 可直接访问目标对象
  • 指向的对象在任何时候都不会被系统回收，（及时抛出OOM）
  • 强引用可能导致内存溢出
